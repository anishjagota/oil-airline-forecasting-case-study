# Oil Prices and Airline Stock Forecasting Case Study

This repository contains the materials for a case study on whether changes in oil prices help improve forecasts of airline ETF (JETS) returns.

## Overview

Airline companies are closely tied to fuel costs, but airline stock performance is also influenced by broader market conditions. This case study asks whether oil prices provide useful predictive information beyond the overall market.

## Main Question

Do oil prices improve forecasts of airline ETF returns after controlling for market performance?
